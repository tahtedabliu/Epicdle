
# EPICDLE

Um jogo inspirado em Wordle onde você precisa adivinhar personagens de **Epic The Musical**.

---

## 🎮 Como jogar

* Digite o nome de um personagem
* Use as sugestões com ↑ ↓ e pressione Enter
* Analise as cores:

🟩 Verde → correto
🟨 Amarelo → parcialmente correto
🟥 Vermelho → incorreto

Você pode tentar quantas vezes quiser!

---

## 🧠 Funcionalidades

* 🔍 Busca inteligente com autocomplete
* ⌨️ Navegação com teclado (setas + Enter)
* 🎯 Sistema de cores inteligente
* 📅 Personagem do dia
* ⏳ Timer para próximo personagem
* 🚫 Bloqueio de respostas repetidas
* 🏆 Tela de vitória
* 📤 Compartilhar resultado estilo Wordle
